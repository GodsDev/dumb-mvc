# dumbMVC

not even an MVC framework 


## features

* none

## Installation

via composer. include this snippet to composer.json
```json
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/tomaskraus/dumb-mvc"
        }
    ],
    "require": {        
        "tomaskraus/php-includer": "@alpha"
    },
```
